# Beyond Good and Evil: Graphical Presentations

A professional, modern, and insightful digital companion to Friedrich Nietzsche's *Beyond Good and Evil* (Walter Kaufmann translation).

## Features
- Graphical PowerPoint-style presentations for each section
- Incisive summaries faithful to the text
- Key quotes highlighted
- Grok Imagine custom visuals (5 slides per section for most)
- Prompts for all generated images included for transparency
- Navigation between sections and master index
- Self-contained HTML pages using Tailwind for beautiful dark academia aesthetic

Ready for readers, students, educators, and public use. Deployed via GitHub Pages.
