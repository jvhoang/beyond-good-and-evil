## Beyond Good and Evil - Graphical Presentations

Extensive summarized and beautiful PowerPoint-like graphical presentations for each of the 296 sections (plus Preface and Aftersong) of Friedrich Nietzsche's *Beyond Good and Evil*, translated by Walter Kaufmann.

**Live Site:** [GitHub Pages](https://jvhoang.github.io/beyond-good-and-evil/)

**Main Embed for GoDaddy:** See `index.html` - copy the content into a Custom HTML section on your finalworth.com/beyond-good-and-evil page.

### Structure
- `index.html`: Main table of contents with search, links to all section presentations.
- `sections/`: Individual HTML presentations for each section (e.g., `001.html`).
- Images generated with Grok Imagine for each 'slide' in the presentations.

### How to Use
- Embed the `index.html` content on GoDaddy for the main page.
- Each section link opens a self-contained, beautiful presentation with summaries and custom generated images.
- All content committed to this repo for version control and hosting.
